# tzap v0.36 Corpus Tracker

Status: evidence tracker
Primary spec: `specs/tzap-format-revisedv36.md`, section 28.1

This document tracks every named corpus case in v0.36 section 28.1. It is not a
conformance claim by itself. A case is `covered` only when the current repo has
reviewable positive evidence and mutation or negative evidence for the case as
described. `partial` means some evidence exists but an implementation mode,
scale case, or mutation remains. `missing` means the required case has no real
fixture/test yet or the underlying feature is absent. `deferred` means the case
is intentionally outside the current release gate and must not be advertised as
supported.
